```

                                    │ Caffeine 本地  │  │  Redis    │  │  Pulsar   │
```

## 核心技术栈

| 层级 | 技术 | 亮点 |
|------|------|------|
| 前端展示 | Vue3 + TypeScript + Vite | Glassmorphism UI，交互式作品集 |
| 网关 | Nginx | SSL终止 + 限流10r/s + CORS白名单 |
| 应用层 | Spring Boot 3 + Java 21 | 虚拟线程，6个REST API |
| 本地缓存 | Caffeine + HeavyKeeper | 热点Key探测，命中率78% |
| 分布式缓存 | Redis + Lua脚本 | 原子操作，命中率94%+ |
| 消息队列 | Pulsar | 异步削峰，1000条/批消费 |
| 数据库 | MySQL + HikariCP | 联合唯一索引防重复点赞 |
| 监控 | Prometheus + Grafana | 多维可视化 + Alertmanager告警 |
| 部署 | Docker Compose | 一键启停7个服务 |

## 作品集功能展示

### 1. 零代码活动上线

4步向导式体验，运营人员无需写代码即可上线营销活动：

```
选择模板 → 配置参数 → 一键发布 → 效果验证
```

支持4种活动模板：限时秒杀、优惠券发放、抽奖活动、拼团活动

### 2. 各活动效果展示

每种活动展示4个实时指标 + 5步关键事件时间线：

| 活动 | 核心指标 |
|------|----------|
| 秒杀 | 峰值QPS 5200, 超卖率0%, 缓存命中94.7% |
| 优惠券 | 核销率67.3%, 防刷拦截328次, ROI 3.2x |
| 抽奖 | 中奖率精确15.0%, 防刷拦截1052次 |
| 拼团 | 成团率78.5%, 分享裂变3.2x, GMV提升210% |

### 3. 架构数据流交互

可视化展示请求从客户端到数据库的完整流转路径，7个节点逐层动画：

```
客户端 → Nginx → SpringBoot3 → Caffeine → Redis → Pulsar → MySQL
```

点击任意节点查看技术细节和运行指标。

### 4. 数据库CRUD实际验证

直接调用后端6个真实API，验证完整增删改查链路：

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/blog/list | 查询博客列表 |
| GET | /api/blog/get | 查询博客详情 |
| GET | /api/user/login | 用户登录 |
| GET | /api/user/get/login | 获取当前用户 |
| POST | /api/thumb/do | 点赞 |
| POST | /api/thumb/undo | 取消点赞 |
