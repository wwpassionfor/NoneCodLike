# 点赞请求处理流程图

```mermaid
sequenceDiagram
    participant User as 用户
    participant Frontend as 前端(Vue3)
    participant Nginx as Nginx网关
    participant Controller as ThumbController
    participant Service as ThumbService
    participant Lua as Redis Lua脚本
    participant Redis as Redis
    participant CacheManager as CacheManager
    participant Caffeine as Caffeine本地缓存
    participant HeavyKeeper as HeavyKeeper热点探测
    participant Pulsar as Apache Pulsar
    Consumer as ThumbConsumer
    DB as 数据库(TiDB/MySQL)
    
    User->>Frontend: 点击点赞按钮
    Frontend->>Nginx: POST /api/thumb/do
    Nginx->>Controller: 转发请求
    Controller->>Service: doThumb(request)
    
    alt MQ模式 (生产环境)
        Service->>Lua: 执行THUMB_SCRIPT_MQ脚本
        Lua->>Redis: HEXISTS检查是否已点赞
        alt 未点赞
            Lua->>Redis: HSET标记用户已点赞
            Lua-->>Service: 返回成功(1)
            Service->>Pulsar: 发送ThumbEvent(INCR)
            Pulsar-->>Service: 异步确认
            Service-->>Controller: 返回true
        else 已点赞
            Lua-->>Service: 返回失败(-1)
            Service-->>Controller: 抛出异常"用户已点赞"
        end
        
        Pulsar->>Consumer: 批量推送消息(1000条/批)
        Consumer->>DB: 批量更新blog.thumbCount
        Consumer->>DB: 批量插入thumb记录
        
    else Redis直连模式 (优化方案)
        Service->>Lua: 执行THUMB_SCRIPT脚本
        Lua->>Redis: 检查+写入临时计数+标记状态
        Lua-->>Service: 返回结果
        Service-->>Controller: 返回结果
        
    else 本地缓存模式 (基础方案)
        Service->>CacheManager: get(userThumbKey, blogId)
        CacheManager->>Caffeine: 查询本地缓存
        alt 本地缓存命中
            Caffeine-->>CacheManager: 返回缓存值
            CacheManager->>HeavyKeeper: 记录访问(+1)
        else 本地缓存未命中
            CacheManager->>Redis: opsForHash().get()
            Redis-->>CacheManager: 返回Redis值
            CacheManager->>HeavyKeeper: add(key, 1)
            alt 是热Key
                HeavyKeeper-->>CacheManager: isHotKey=true
                CacheManager->>Caffeine: put(compositeKey, value)
            else 不是热Key
                HeavyKeeper-->>CacheManager: isHotKey=false
            end
        end
        CacheManager-->>Service: 返回点赞状态
        Service->>DB: 编程式事务更新
    end
    
    Controller-->>Frontend: BaseResponse<Boolean>
    Frontend-->>User: 更新UI显示
```
